# LaquaiLib.Threading.CronThreadingPrimitives.Internal

Internal helpers shared across the `LaquaiLib.Threading.CronThreadingPrimitives` assemblies. **Not intended for direct use.**

## Contents

- **`CronHelpers`** — static helper that parses and validates arrays of cron expression strings, trying both 5-field (standard) and 6-field (with seconds) formats via [Cronos](https://github.com/HangfireIO/Cronos).
- **`VoidStruct`** — reserved internal value type placeholder.
